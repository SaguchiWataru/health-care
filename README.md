# 健康管理表
## 概要
新型コロナウイルスの感染拡大を防ぐために、いつでもどこからでも健康状態を登録できるWebサイトを作成しました。  
実際のWebサイトは、こちらからご覧ください。[https://saguchi-wataru.azurewebsites.net/health-care](https://saguchi-wataru.azurewebsites.net/health-care)
